# 平均成本法（均摊成本）持仓核算

用户偏好平均成本法（移动平均），非 FIFO。

## 公式

```
加仓后均价 = (原 total_cost + 买入成本) / (原 shares + 新股数)
减仓后 total_cost = 原 total_cost - 卖出实收（price × shares - commission）
减仓后均价 = 新 total_cost / 剩余股数
卖亏了 → 剩余均价上升，卖赚了 → 剩余均价下降
```

## 示例

```
初始: 8股, total_cost=$7,638.02, 均价=$954.75

卖出 2 股 @ $969（实收 $1,938，无佣金）:
  新 total_cost = $7,638.02 - $1,938 = $5,700.02
  剩余: 6股, 均价 = $5,700.02 / 6 = $950.00（卖赚了，均价↓）

卖出 2 股 @ $900（实收 $1,800）:
  新 total_cost = $5,700.02 - $1,800 = $3,900.02
  剩余: 4股, 均价 = $3,900.02 / 4 = $975.01（卖亏了，均价↑）

加仓 2 股 @ $1,000（支出 $2,000）:
  新 total_cost = $3,900.02 + $2,000 = $5,900.02
  新均价 = $5,900.02 / 6 = $983.34

清仓 6 股 @ $1,100（实收 $6,600）:
  新 total_cost = $5,900.02 - $6,600 = -$699.98（负成本，已收回本金+盈利）
```

## 为何卖出要减实收而非减成本价

```
卖出前：8股 × $954.75 = $7,638（你总共花了这么多钱）
卖出 2股@$969：拿回 $1,938
卖出后：总投资还剩 $7,638 - $1,938 = $5,700
剩余 6股：$5,700 / 6 = $950/股
```

你的总投资减少了卖出拿回的现金，剩余成本就是真实套牢的钱。卖赚了你口袋里钱多了，套在这只股上的钱自然变少了。

## 与 FIFO 的区别

| | 平均成本法 | FIFO |
|---|---|---|
| 减仓扣哪个批次 | 按总成本比例 | 按最早买入批次 |
| 剩余成本 | 总成本 - 卖出实收 | 总成本 - 卖出批次成本 |
| 适用场景 | 国内券商（同花顺/富途） | 美股 IRS 税务最优 |
