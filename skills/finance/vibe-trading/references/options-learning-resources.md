# Options Learning Resources

Curated external resources for learning stock options trading from scratch. Broken down by format and skill level.

## Free Web Resources

### The Options Playbook — best for beginners
- **Site:** optionsplaybook.com
- **Structure:** Strategies grouped by experience level: Rookies → Veterans → Seasoned Veterans → All-Stars
- **Also has:** Options Basics (Call/Put/Greeks/Volatility/Open Interest), Rookie's Corner (5 common mistakes, rolling positions), Glossary
- **Format:** One-page per strategy, clean UI, no signup

### Investopedia Options Tutorial
- **Options Basics Tutorial:** investopedia.com/options-basics-tutorial-4583012
- **5-Step Guide:** investopedia.com/articles/active-trading/040915/guide-option-trading-strategies-beginners.asp
- **Complete library:** investopedia.com/options-and-derivatives-trading-4689663

### Tastytrade Learn
- **Site:** tastytrade.com/learn
- **Focus:** IV (implied volatility), Greeks, probability-based thinking
- **Format:** Free video library — they are a dedicated options broker, content is practical

### Reddit Community
- **r/options:** reddit.com/r/options/ — high-quality discussion, search before asking

## Books

| Book | Author | Level | Notes |
|------|--------|-------|-------|
| The Options Playbook | Brian Overby | Beginner | Strategy encyclopedia, one-page per strategy |
| Understanding Options | Michael Sincere | Beginner | Friendliest intro, great for absolute beginners |
| Options as a Strategic Investment | Lawrence McMillan | Advanced | The "bible" — 1000+ pages, reference-grade |
| Option Volatility & Pricing | Sheldon Natenberg | Advanced | The definitive work on IV and pricing |

## Recommended Learning Path

1. **Options Basics** (Options Playbook) — Call/Put/Strike/Expiry/Premium
2. **Greeks** (Tastytrade videos) — Delta/Gamma/Theta/Vega — most important concept
3. **Single-leg strategies:** Covered Call → Cash-Secured Put → Long Call → Long Put
4. **IV & volatility** — Tastytrade IV content before attempting spreads
5. **Multi-leg strategies:** Vertical Spreads → Iron Condor → Butterflies
6. **Paper trade** — Thinkorswim paper trading, test strategies risk-free

## Web Search Strategy (when the user asks for learning resources)

The user prefers that you **search the web** for existing resources rather than create new skills or generate fabricated content. Use DuckDuckGo HTML search (not Google — less blocking):

```bash
curl -s "https://html.duckduckgo.com/html?q=search+terms" -H "User-Agent: Mozilla/5.0"
```

For sites that block (Google, most search engines), try direct access to known resource sites:
- Wikipedia (en.wikipedia.org) — raw content via curl
- Known finance sites directly (optionsplaybook.com, investopedia.com, tastytrade.com)
