# 军舰通过霍尔木兹海峡 — 各国概率评估

## 通用原则

- **霍尔木兹沿岸国家**（阿曼、阿联酋、卡塔尔、科威特、巴林、沙特、伊朗）→ 军舰日常穿越海峡，概率高
- **有波斯湾常驻军事基地的国家**（美国、英国）→ 例行任务概率中高
- **偶发参与CMF的国家**（韩国、巴基斯坦、希腊）→ 2-3天内几乎没有理由专门部署
