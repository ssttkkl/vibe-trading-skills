# 结算范围盘波动率感知分析

> 价格范围盘（如 Gold $3800-$4200、BTC $50K-$67.5K）是 Polymarket 上常见类型。AI 容易只看当前价格就下结论，忽略波动率风险。

## 核心原则

**当前价格在范围内 ≠ 到期时也在范围内。** 高波动率资产可能在几天内大幅突破范围。

## 分析框架

### Step 1：获取波动率数据

用 `volatility_dashboard.py`（技能 `scripts/` 下）自动输出：

```
BTC 30d范围: $58,189 - $74,091  (30d变化: -17.6%)
ETH 30d范围: $1,523 - $2,038  (30d变化: -20.6%)
XAU/USD: $4,091
⚠️ 黄金近期波动率极高（本月振幅 $3800-$5000，约24%）
```

关键指标：
- **30d范围** — 资产近一个月波动到底有多大
- **30d变化%** — 趋势方向
- **24h变化** — 短期动量
- **月振幅%** — 波动率定性判断（>15% = 高波动）

### Step 2：评估范围安全边际

```
结算范围：$3,800 - $4,200
当前价格：$4,091
30d范围：$3,800 - $5,000（波动24%）

到上限距离：($4,200 - $4,091) / ($5,000 - $3,800) × 100 = 8.3% 的范围宽度
到下限距离：($4,091 - $3,800) / ($5,000 - $3,800) × 100 = 22.5% 的范围宽度

推理：当前价靠近上限（只占范围宽度约8%），
      以24%月波动率来看，3天内涨破$4,200完全可能
      → 买Yes @ 87.6¢ 风险很大
```

安全边际经验：
- 当前价格在范围中间1/3 → 相对安全
- 当前价格在范围边界20%内 → 高风险（高波动资产尤其）
- 30d波动范围完全包含结算范围 → 价格可能在到期前反复进出范围

### Step 3：检查到期前宏观事件

用 `web_search` 搜索未来N天的重要宏观事件：

| 资产 | 关键事件 | 影响 |
|:----|:---------|:----|
| 黄金 | Fed利率决议、CPI/PCE、非农、地缘冲突 | 央行利率预期直接影响金价 |
| BTC | ETF资金流、SEC监管、Mt.Gox赔付 | 供需冲击 |
| 石油 | OPEC会议、EIA库存、地缘（霍尔木兹） | 供应端冲击 |
| 股指 | Fed会议、财报季、GDP数据 | 宏观风险偏好 |

**如果到期前有重大事件且当前价在范围边缘 → 高波动率事件 = 大风险**

### Step 4：综合判断

| 信号 | 判断 | 可投? |
|:----|:----|:-----|
| 价格在范围中间 + 低波动 + 无宏观事件 | 🟢 安全 | ✅ |
| 价格在范围中间 + 中波动 | 🟡 可接受 | ⚠️ 小资金 |
| 价格在范围边界 + 高波动 | 🔴 高风险 | ❌ |
| 价格在范围外 + 靠波动进场 | 🔴 彩票 | ❌ |
| 已到期/今天到期且当前价在范围内 | 🟢 确定 | ✅ |

## 典型案例

### On: Gold $3800-$4200 June settle（3天到期）

```
当前：$4,091 | 30d范围：$3,800-$5,000 | 波幅24%
Yes @ 87.6¢ (ROI +14.2%)

A: 价格在范围内但靠近上限($4,200)
B: 波动率24% → 3天内波动±2%很正常
C: 6月29日有核心PCE数据
结论：Yes @ 87.6¢ 风险高，不推荐
```

### On: BTC >$60K today（当天到期）

```
当前：$60,369 | 今天到期
Yes @ 90.5¢ (ROI +10.6%)

A: 当前已>$60K
B: 今天到期，今天是结算日
C: 只要不瞬间暴跌就已经结算
结论：Yes @ 90.5¢ 送钱，%99 确定
```

### On: ETH dip to $1,400 in June（4天到期）

```
当前：$1,581 | 30d范围：$1,523-$2,038 | 波幅-20.6%
No @ 94.5¢ (ROI +5.8%)

A: $1,581 距 $1,400 有 $181 空间(+11.5%)
B: 但30d低点$1,523 → 曾到过$1,523
C: 从$1,523到$1,400还差$123(约8%再跌)
D: 4天内从$1,581再跌11.5%到$1,400？可能但不确定
结论：No @ 94.5¢ 较安全但不完全确定
```

## 从实时行情判断结算盘

有跨到期日子市场时，可用跨到期日价格差反推当前状态：

```
Strait of Hormuz returns to normal by:
├── 6/30到期: Yes @ 4.75¢
└── 7/31到期: Yes @ 49.5¢

推理：如果当前通行已接近正常，6/30不会只有4.75¢
      → 当前通行量大概率远低于正常水平
      → 6/30 No 是安全盘
```

## 统计波动率速查表（经验值）

| 资产 | 典型月波动率 | 安全分析窗口 |
|:----|:----------:|:-----------|
| 黄金 | 10-25% | 3天内±2%正常，一周±5%正常 |
| BTC | 20-40% | 一天±5%正常，一周±15%常见 |
| ETH | 25-45% | 跟随BTC，波动更大 |
| SOL | 30-60% | 高波动，谨慎 |
| WTI原油 | 8-20% | 受OPEC/EIA事件驱动 |
| S&P 500 | 5-15% | 相对平稳，关注宏观事件 |
